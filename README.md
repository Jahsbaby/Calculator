## Features

```
- Add
- Multiply
- Subtract
- Divide
- Square
- Square Root
- Power of n
- Natural exponential function
- Factorial of n
- Trigonometric operation
- Cuberoot
-nPr[Permutation of n elments]
-nCr[Number of combinations for given set]
```
